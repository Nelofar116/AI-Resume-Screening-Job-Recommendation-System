# AI Resume Screening & Job Recommendation System

## Overview
A web application that automates resume screening and recommends suitable job roles based on candidate skills.

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- Apache PDFBox
- Maven

## Features
- Upload PDF resumes
- Extract text from resumes using Apache PDFBox
- Identify candidate skills
- Recommend suitable job roles
- REST API integration
- User-friendly web interface

## Project Structure
- Controllers
- Services
- Models
- HTML Frontend Pages

## How to Run
1. Clone the repository
2. Open the project in VS Code or IntelliJ IDEA
3. Configure Maven dependencies
4. Run DemoApplication.java
5. Open the application in your browser
